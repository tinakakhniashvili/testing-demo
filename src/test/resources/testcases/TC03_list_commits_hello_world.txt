Test Case ID: TC03
Title: List commits for repository (octocat/Hello-World)
Endpoint: GET https://api.github.com/repos/octocat/Hello-World/commits
Auth: Not required

Preconditions:
- GitHub API is reachable.

Steps:
1) Send GET request to https://api.github.com/repos/octocat/Hello-World/commits
2) Observe response status and JSON body.

Expected Result:
- Status code: 200 OK
- Content-Type contains "application/json"
- Response body is a JSON array
- First element contains:
  - sha (string)
  - commit.author.name (string)
  - commit.author.date (string)
