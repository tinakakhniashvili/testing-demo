Test Case ID: TC05
Title: Get rate limit status
Endpoint: GET https://api.github.com/rate_limit
Auth: Not required (works with or without token)

Preconditions:
- GitHub API is reachable.

Steps:
1) Send GET request to https://api.github.com/rate_limit
2) Observe response status and JSON body.

Expected Result:
- Status code: 200 OK
- Content-Type contains "application/json"
- Response body contains:
  - resources (object)
  - rate.limit (number)
  - rate.remaining (number)
  - rate.reset (number)
